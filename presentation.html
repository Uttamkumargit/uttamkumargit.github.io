Importing different file in R

  • Usually we will be using data already in a file that we need to read into R in order to work on it. R can read data from a variety of file formats—for example, files created as text, or in Excel, SPSS or Stata.

  • To read an entire data frame directly, the external file will normally have a special form.

  • We will mainly be reading files in text format .txt or .csv (comma-separated, usually created in Excel).
